API desenvolvida para a disciplina de Back-End: Web Services, composta por autenticação baseada em JWT, controle de permissões para Administrador e Usuário e múltiplos módulos CRUD implementados pelos integrantes da equipe. Cada módulo oferece operações completas, incluindo listagem, busca por ID, criação, atualização e exclusão, com persistência em banco de dados e integração entre componentes desenvolvidos pelo grupo. A aplicação também possui um sistema global de tratamento de erros com mensagens padronizadas e inclui testes automatizados nos níveis de Controller e Service.
